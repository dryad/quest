Class code_menu

Class Overview
Class Overview | Inherited Properties, Constants, and Methods | Property Summary | Properties Detail | Method Summary | Methods Detail

Handles that lovely list that goes down the side of the page

Defines a process for hooking into the menu entry, static function <class_name>_menu which takes &$menu as a parameter, returns a string to be appended to the menu label, and can add to the class variables $top and $bottom for new-mail style fun stuff.
