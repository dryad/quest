Class code_common

Class Overview
Class Overview | Descendants | Property Summary | Properties Detail | Method Summary | Methods Detail

Base class for modular design

code_common is the base class for 90% of the other classes in the project - it only handles pretty much universal concepts such as database connections, the skin, the player etc - shouldn't be anything specific to one part.
